при открытии приложения:

показываем СТРАНИЦУ "Список превью постов:"
загрузить посты по api /users и отрисовать их:
превью поста содержит title целиком и body, обрезанный до 25 символов

чтоб посчитаь количество комментариев, нужно сделать что ?
чтоб найти имя автора поста, нужно сделать что ?
рейтинг отрисовываем рандомно

при клике на кнопку "Подробней..." у поста:
загрузить комментарии к посту по api /comments?postId=42 и отрисовать их
показываем СТРАНИЦУ "Подробный пост:"
Подробный пост содержит title целиком и body целиком плюс список комментариев

при клике на кнопку имя автора поста у поста:
загрузить посты от данного автора по api /posts?userId=4 и отрисовать их
показываем СТРАНИЦУ "Список кратких постов:" от данного автора
